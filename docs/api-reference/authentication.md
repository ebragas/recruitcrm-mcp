# Authentication

To access the API you will need your API Token from the Admin Settings.
All calls to the API require an Authorization header in the following format:

> Authorization: Bearer <YOUR\_API\_TOKEN>

To get your API Token, please follow these steps.

- Log in to Recruit CRM as an admin for your organization
- Go to [Admin Settings](https://app.recruitcrm.io/admin-settings)
- Navigate to [API & Integrations Section](https://app.recruitcrm.io/api-and-integrations)
- Click on Copy next to your API Token to copy it to you your clipboard
