# Custom Field Search

List of Supported Filter Types:-

| Filter Type | Field Types | Filter Value Required |
| --- | --- | --- |
| **equals** | text, longtext, number, date, dropdown, multiselect, email, phonenumber | Yes |
| **not\_equals** | text, longtext, number, date, dropdown, multiselect, email, phonenumber | Yes |
| **contains** | text, longtext, email, phonenumber | Yes |
| **not\_contains** | text, longtext, email, phonenumber | Yes |
| **available** | text, longtext, number, date, dropdown, multiselect, email,phonenumber,file | No |
| **not\_available** | text, longtext, number, date, dropdown, multiselect, email,phonenumber,file | No |
| **greater\_than** | number, date | Yes |
| **less\_than** | number, date | Yes |
| **yes** | checkbox | No |
| **no** | checkbox | No |
