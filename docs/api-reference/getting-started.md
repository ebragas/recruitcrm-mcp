# Getting Started

At Recruit CRM, We are on a mission to help recruitment businesses grow with technology.

Our API is built to empower any developer or team to customize and get the most out of Recruit CRM. Our API gives you access to Add & Edit Candidate, Company, Contact and Job data.

The Recruit CRM API is organized around REST and is designed to have predictable resource-oriented URLs. You can use many standard HTTP features, methods (POST, GET, PUT) and error response codes.

### Base URL

All API calls are made using [https://api.recruitcrm.io](https://api.recruitcrm.io/) and all responses return JSON.

### Featured APIs

- [Candidates](https://docs.recruitcrm.io/docs/rcrm-api-reference/url) API endpoints allow you to list,create,update,delete and assign candidates.
- [Jobs](https://docs.recruitcrm.io/docs/rcrm-api-reference/url) API endpoints allow you to manage your jobs.
